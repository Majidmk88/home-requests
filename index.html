How to get my Firebase URL? ›
Quick Firebase Setup (free, 2 min):

1. Go to console.firebase.google.com
2. Click "Add project" → name it "homely" → Continue
3. Click "Realtime Database" → Create database
4. Choose "Start in test mode" → Enable
5. Copy the URL shown (looks like https://homely-xxxxx-default-rtdb.firebaseio.com)
6. Paste it above and tap Connect

Do this once — then share the same app URL with Noura and the maid. All three phones will sync instantly.
